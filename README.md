# kubix-core
Based on Java and LWJGL, this library will provide core kubix engine functionality to create quality game.

## Requirements
* Java (JDK 17+)
* Apache Maven
* Git

## Build
```
mvn clean install
```

## Start
```
java -jar target/kubix-core-{POM_VERSION}.jar
```
